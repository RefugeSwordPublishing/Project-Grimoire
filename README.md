# Project Grimoire

A semi-idle RPG built in Unity (C#) with a Supabase backend. Mobile-first with a planned Steam release.

## Docs
All design documentation lives in `/docs`. Read before implementing anything.

## Tech Stack
- Engine: Unity 2022.3 LTS or Unity 6
- Language: C#
- Backend: Supabase
- Notifications: Firebase Cloud Messaging
- Monetization: Unity IAP
- Analytics: GameAnalytics
